
Lira-Kurdi ($LKURD)

Lira-Kurdi is a new cryptocurrency project designed for the Mesopotamian region, built on TON (The Open Network) technology.
It represents the spirit of innovation, connecting cultures, finance, and technology under one ecosystem.

🌍 Supported Regions:
	•	Turkey | Syria | Iraq | Iran
	•	Active as a regional digital currency for Mesopotamia.

🚀 Core Projects:
	•	Profile System — Decentralized identity management
	•	Wallet — Secure TON-based $LKURD crypto wallet
	•	Airdrop — Community-driven token distribution
	•	NFT Platform — Create, buy, and sell Mesopotamian-themed NFTs
	•	Games & Apps — Play-to-earn integrations and lifestyle applications
	•	Finance Tools — Crypto payments, staking, and donation systems

💰 Market Capacity Plan:
	•	NFT Ecosystem: Mesopotamian art & cultural collections
	•	TON Token Integration: Seamless $LKURD transactions
	•	Donation Platform: Support regional development through crypto

🔗 Technologies Used:
	•	TON Blockchain
	•	JavaScript / HTML / CSS
	•	Telegram Mini Apps
